Network-connected Digital Frame Platform

NetFrames is a lightweight, connected display platform that allows digital picture frames or embedded screens to automatically fetch and display images from a central local server. The entire platform is built with .NET: server (ASP .NET Core Web API), admin portal (.NET Blazor Web App), and client (Meadow F7 app).

Bus Stop Companion App

Created an interactive bus stop mobile application using .NET MAUI, that talks with Vancouver's transit API to provide real-time bus information at any stop. Users can view arrival times, set reminders, and get notifications for their favorite routes.

Pond Monitoring Companion app

Built a companion mobile (.NET MAUI) app that connects to a smart pond aerator over Bluetooth Low-Energy (BLE) and provides basic management and HMI functions such as viewing current operating conditions, setting the operational schedule, and basic commissioning.
